git config --global user.name blabla - прописуємо ім'я яке буде відображатися в колонці автора git config --global user.email blabal@bla - пошта яка відображається в інформації про виконаних вами комітів. git init - ініціалізація гіта в данній папці git status - перегляд змін і створення коммітів git add - вказуєм файл який хочемо закомітити. git commit - створюємо зліпок файла (зберігаємо зміни). git log - перегляд історії зміни файла. git checkout - переключити на вибрану гілку і обновити робочу теку до її стану. git branch - видалити дану гілку. git push - завантажуєм всі зміни локальної гілки в віддалений репрозиторій. git pull - завантажуєм історію із віддаленого репрозиторія і обєднуємо її з локальною. git merge - вносимо зміни вказаної гілки в дану гілку. git clone - завантажити репрозиторій разом із його історією змін. .gitignore - вказуємо що буде ігнорувати гіт. рядки чи файли. ігнорує - не слідкує за змінами. git revert - відміна змін...
